## Reading Rule

This index records topic-level process provenance only. It must not become a
dated closeout ledger, verifier transcript index, render-output list, pilot
artifact manifest, worktree log, release/currentness claim, or active gap plan.

If a historical item still matters today, fold the durable rule into
`docs/status.md`, `docs/active/bookforge-ideal-state-gap-plan.md`,
`docs/docs_portfolio_consolidation.md`, `docs/evidence/README.md`, core docs,
contracts, agent pack files, source, validators, owner receipts, or typed
blockers. Keep only the compressed provenance here.
