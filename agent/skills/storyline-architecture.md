# Storyline Architecture Skill

Use this skill when shaping a book-length project before drafting chapters.

Working policy:

- Read source refs, owner intent, audience, and comparable book refs first.
- Convert the intent into a stable premise, argument path, chapter thesis chain, and evidence map.
- Preserve a single style contract across later chapters.
- Identify where evidence is thin, duplicated, or misordered.
- Return typed blockers for missing source, owner, style, or audience refs.

This skill does not authorize final content acceptance, publication readiness, or artifact export.
