# Storyline Architecture Stage

Stage id: `storyline-architecture`
Action ref: `shape-storyline`

This stage owns the first half of the OPL BookForge workflow. It organizes the story before drafting: promise, reader path, argument arc, chapter chain, evidence placement, motif, pace, and style rule.

The stage should produce enough structure for chapter writing to proceed without re-deciding the book each time. It can route back for missing audience, corpus, publication form, or owner preference refs.
