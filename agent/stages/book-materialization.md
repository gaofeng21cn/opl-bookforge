# Book Materialization Stage

Stage id: `book-materialization`
Action ref: `materialize-book`

This stage owns the second half of the workflow. It turns the storyline into book deliverable refs: chapters, illustration plan, tables, style pass, wording pass, layout checks, and handoff.

Quality control covers content consistency, writing style consistency, terminology, source grounding, illustration/table fit, typography, pacing, and owner decisions. The stage returns typed blockers when manuscript body, data refs, image rights, layout target, or owner gate evidence is missing.
